It may also be necessary for format certain content into lists. Of the top of my head, that could include potential features to impliment in the future, various bug fixes, lists of completed tasks, or various ideas for implimenting a certain feature. In any of those cases it would be for,atted into an unordered list as follows:

  • List Item #1
  • List Item #2
  • List Item #3

When describing an order of events which took place, like describing the order of game events, it would be more fitting to use an ordered list. Those will appear as follows:

  1. This is the first thing that happens.
  2. This is the second.
  3. This is last.
