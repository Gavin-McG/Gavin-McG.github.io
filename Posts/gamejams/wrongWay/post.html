Wrong Way!!!

Concept

Wrong Way!!! is a project created for the 2024 Global Game jam from the prompt “Make me laugh”. The game is a 2-player local multiplayer where the player must throw frisbees into the crowd to score points as members of the audience laugh at their peers’ misfortune.

This project used several Unity features which I hadn’t previously taken advantage of, such as custom skyboxes, trails, and changing global physics properties. I also had to learn more about using materials and shaders to allow a gradient of skin tones both in regards to the players and members of the audience.

Controls

  • JoyStick to aim throw
  • Hold X/A to charge throw
  • Release X/A to release throw

Requirements

  • 2 Controllers
