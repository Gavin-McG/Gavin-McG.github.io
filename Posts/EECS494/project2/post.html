Gluten Gunning

Information

Student Name: Gavin McGinness

This project was created solo for EECS 494 Project 2 over the course of 2 weeks. The game features a combat between two players who must collect bread, litter it with toppings, before shooting the resulting toast at their opponent. Each topping has a unique effect. Beyond the features of the game, in completeing this project I gained extra experinece in the following tools within Unity:

  • Baked Lighting
  • URP
  • Input System
  • Texturing

Requires at least 1 Controller + Keyboard&Mouse
